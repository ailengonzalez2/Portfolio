COFF

UI Development

Technologies: Figma, Nuxt, Nuxt.UI, TailwindCSS, Web3 Modal

This decentralized application was created, as part of a hackathon to revolutionize the coffee purchasing experience using NFT technology. Users can buy and sell tokens, COFF, which can be acquired anytime, anywhere. These tokens can be redeemed at registered stores, providing a faster, safer, and more transparent experience.
